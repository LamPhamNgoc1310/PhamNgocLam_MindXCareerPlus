Exercise 2: Input tags

type text will receive text in any format

type password will receive text in any format and hide them under black circle form

type number only receive integer

type file will receive file dragged in or input from file explorer

type checkbox will receive arguments like "True" or "False" in the form of ticked and unticked box

type radio will create a set with multiple choices, the user can only select 1 from the set
